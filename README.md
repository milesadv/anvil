# Self Made Good

Ora et Labora. Luxury through labour.

## About

Self Made Good is a luxury menswear brand for men who have earned their success through discipline, work, and craft. This is the brand's digital presenceâ€”a cinematic, immersive experience featuring an audio-reactive visualizer.

## Features

- **Audio Visualizer**: Upload audio and watch it transform into reactive particle systems
- **Torus Shader**: Organic, iridescent particle blob in the idle state
- **Frequency Filter**: Mouse-controlled low/high pass filter in frequency mode
- **Email Capture**: Join the list for the first drop (March 2026)

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **3D Graphics**: React Three Fiber + Three.js
- **Styling**: Tailwind CSS v4
- **Fonts**: Geist Sans, Geist Mono, Cormorant Garamond

## Getting Started

### Prerequisites

- Node.js 20+
- pnpm

### Installation

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Scripts

```bash
pnpm dev          # Start development server
pnpm build        # Build for production
pnpm start        # Start production server
pnpm lint         # Run ESLint
pnpm type-check   # TypeScript type checking
```

## Deployment

Deploy on Vercel:

vercel
